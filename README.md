====== Sistema de Biblioteca======

Este programa fue creado en Python para registrar y administrar libros de una biblioteca.

Permite:
agregar libros, buscarlos, eliminarlos, actualizar su disponibilidad y mostrar la lista completa. 
Cada libro guarda su título, cantidad de copias, días de préstamo y si está disponible o no.

También incluye validaciones para evitar datos vacíos o incorrectos, como títulos sin texto, copias negativas o días de préstamo menores a 1.
