=====Sistema de Biblioteca=====

Este programa fue creado en Python para registrar y administrar libros de una biblioteca.

Permite:
agregar libros, buscar libros, eliminarlos, actualizar su disponibilidad y mostrar la lista completa. Cada libro guarda su título, cantidad de copias, días de préstamo y si está disponible o no.

 =====Instrucciones de uso=====

1. Ejecutar el archivo del programa en Python.
2. Al iniciar, aparecerá un menú con varias opciones.
3. Escribir el número de la opción que se quiere usar.
4. Para agregar un libro, ingresar el título, las copias disponibles y los días de préstamo.
5. Para buscar o eliminar un libro, escribir el título del libro.
6. Para ver todos los libros, elegir la opción “Mostrar libros”.
7. Para terminar el programa, elegir la opción “Salir”.

El programa valida que no se ingresen títulos vacíos, copias negativas o días de préstamo menores a 1.
